# PNC Electrostatic Fix
PneumaticCraft's Electrostatic Compressor summons a lightning bolt only when certain conditions are met.

It also does not allow regular lightning bolts, be it summoned via command or other mods' blocks or mechanics, to trigger the Compressor to work.

This mod makes the Electrostatic Compressor work with any lightning bolt. No more waiting around for a random lightning strike.

Does this break balance? Yeah probably lol
